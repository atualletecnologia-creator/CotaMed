# Correção: busca ao continuar cotação salva

- Ao restaurar um rascunho, o Banco de Preços é carregado novamente.
- A busca manual volta a listar produtos para todos os itens.
- O campo de busca tenta recarregar os produtos caso o banco ainda esteja vazio.
- A busca manual aceita variações parciais do descritivo.
