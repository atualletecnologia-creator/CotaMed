CotaMed - Biblioteca Inteligente de Documentos

Nova versão da aba Documentação:
- Adicionada aba interna "Biblioteca inteligente".
- Biblioteca padrão já preenchida com regras do trecho de edital enviado.
- Cada regra possui:
  categoria
  documento
  palavras-chave
  termos equivalentes
  termos de dispensa/condição
  obrigatoriedade
  ordem
  observações
- Permite criar, editar, excluir e restaurar regras.
- Tudo salvo localmente no navegador/computador.
- O analisador do edital usa a biblioteca para montar:
  pastas organizadas
  documentos não encontrados
  documentos não solicitados
  termos condicionais/dispensas
  relatório de conferência.
