Correção Licitações - quantidade e unidade

Alteração:
- Importador agora reconhece mais nomes de coluna para quantidade:
  quantidade, quant, qtd, qtde, qde, quantidade_total, qtd_solicitada, qtd_licitada, etc.
- Importador agora reconhece mais nomes de coluna para unidade:
  unidade, unid, un, und, unidade_medida, un_medida, u_m, um, medida, embalagem, apresentação.
- Se a quantidade não vier preenchida, usa 1.
- Se a unidade não vier preenchida, usa UNIDADE.
