CotaMed - Documentação com aprendizado

Atualizações:
- Campo de upload do edital (.pdf ou .txt).
- Extração simples de texto para PDFs com texto embutido.
- PDFs escaneados ainda precisam ter o trecho colado manualmente.
- Botão "Aprender com este edital".
- Gera sugestões de novos termos não reconhecidos.
- Permite escolher o documento correto e salvar como:
  - palavra-chave
  - termo equivalente
  - termo de dispensa
- O aprendizado é salvo localmente no navegador/computador.
- A análise passa a usar a biblioteca atualizada.

Observação:
A extração de PDF no navegador é simples. Para PDF escaneado ou protegido, cole o texto do edital no campo.
